## Списки

Чтобы добавить ненумерованные списки, нужно списки выделить звездочко (*). Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки необходимо пункты просто пронумеровать. Например, вот так:
1. Элемент 1
2. Элемент 2.
